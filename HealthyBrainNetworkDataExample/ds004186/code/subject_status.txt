The subject_status.csv file contains the following columns:

subject 		- the subject ID
release 		- the release number
isComplete	- 1 if the data could be processed, 0 otherwise
EOCount		- the number of eyes-open events (should be 6)
ECCount		- the number of eyes-closed events (should be 5)
totalEventCount  - the total number of events in the file
isMatExist	- 1 if the MATLAB raw data file exist, 0 otherwise
matSize		- size of the MATLAB file
isMffExist	- 1 if the MATLAB MFF (EGI) data file exist, 0 otherwise
mffSize		- size of the MFF file
